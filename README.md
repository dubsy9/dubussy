# Dubussy's Paradise

A simple web application built with Cloudflare Workers.

## What's in this build

- **AI Chat Assistant** - Integrated n8n chatbot for AI assistance
- **Request System** - Email submission form that sends notifications to Discord
- **Static Site** - HTML/CSS/JS frontend served via Cloudflare Pages

## Tech Stack

- Cloudflare Workers (wrangler)
- n8n chat integration
- Vanilla JavaScript
